Satz von Darboux

1. Satz

Sei \(f:[a,b] \rightarrow \mathbb{R}\) eine differenzierbare Funktion mit o.B.d.A. \(f'(a)) < f'(b)\), so folgt, dass für beliebiges \(c \in (f'(a),f'(b))\) ein \(x_0 \in (a,b)\) existiert mit

\begin{align*} f'(x_0) = c \end{align*}

2. Beweis

Wir wissen, dass \(f\) stetig, ist da differenzierbaren Funktionen stetig sind. Sei \(g(x) \coloneqq f(x) - c\cdot x\), ebenfalls stetig und differenzierbar. Damit nimmt \(g\) auf einem folgenkompakten Intervall ein Minimum bzw.Maximum an (siehe: Satz vom Minimum und Maximum) Insbesondere gilt für hinreichend kleine \(\varepsilon > 0\):

\begin{align*} g'(a) < 0 \Rightarrow g(a) > g(a + \varepsilon) \\ g'(b) > 0 \Rightarrow g(a - \varepsilon) < g(a) \end{align*}

D.h. ein Minimum ist in \((a,b)\) Daraus folgt eine Nullstelle der Ableitung von \(g'\) und es gilt

\begin{align*} 0 = f(x_0) - c \Rightarrow f(x_0) = c \end{align*}

Alternativ: Widerspruch wegen der Wahl von \(c \neq f'(a),f'(b)\)
