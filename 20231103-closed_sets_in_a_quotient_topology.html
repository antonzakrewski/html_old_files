closed sets in a quotient topology

1. Proposition

Let \((X, \mathcal{T})\) be a topological space and \(X/\sim\) for \(\pi: X \rightarrow X/\sim\) the quotient topology. Then a set \(A/\sim \subesteq A\) is closed if and only if \(\pi^{-1}[A/\sim]\) is closed
