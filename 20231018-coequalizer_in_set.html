coequalizer in Set

1. Proposition

Given the category set, sets \(A,B \in \mathrm{Ob}(\mathrm{Set})\) and maps \(\varphi, \psi: A \rightarrow B\), the coequalizer is the set

\begin{align*} CE \coloneqq B / \sim \end{align*}

for the smallest equivalence relation such that \(\pi \circ f(a) = \pi \circ g(a)\) with the usual projection \(\pi\) as morphism 1

2.2. constructive

\begin{align*} b_1 \sim b_2 :\Leftrightarrow \begin{cases} b_1 = b_2 \\ \exists x_1,...,x_n : f(x_i) = g(x_{i+1}), f(x_1) = b_1, f(x_n) = b_2 \\ \exists x_1,x_2 : f(x_1) = f(x_2), g(x_1) = b_1, g(x_2) = b_2 \\ \exists x_1,x_2 : g(x_1) = g(x_2), f(x_1) = b_1, f(x_2) = b_2 \end{cases} \end{align*}
