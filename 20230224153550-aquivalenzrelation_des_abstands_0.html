Äquivalenzrelation des Abstands 0

1. Satz

Sei \((M,d)\) ein semimetrischer Raum, dann ist die Relation

\begin{align*} x ~ y \Leftrightarrow d(x,y) = 0 && \forall x,y \in M \end{align*}

eine Äquivalenzrelation

2. Beweis

2.1. Symmetrie

folgt aus der Symmetrie einer Metrik

2.2. Transitivität

Seien \(x,y,z \in M\), so dass gilt:

\begin{align*} d(x,y) = 0 \\ d(y,x) = 0 \end{align*}

Dann folgt aus der Dreiecksungleichung und Nichtnegativität

\begin{align*} 0 \leq d(x,z) \leq d(x,y) + d(y,x) \leq 0 \\ \Rightarrow d(x,z) = 0 \end{align*}

2.3. Reflexivität

folgt aus Selbstabstand einer Seminorm
