Quantitative Taylor-Approximation

1. Satz

Sei \(f: I \rightarrow \mathbb{C}\) \(n\)-mal differenzierbar (\(n \in \mathbb{N}\)) so, gilt: Füre verschiedene \(x,x_0 \in I\) existiert ein \(x \leq \xi \leq x_0\) mit

\begin{align*} f(x) = T_{x_0,n-1}(x) + \frac{f^{(n)}(\xi)}{n!} (x - x_0)^n \end{align*}

siehe: Lagrange-Restglied

2. Beweis

Es gilt in \(x_0\) für \(1 \leq i \leq n-1\):

\begin{align*} (f - T_{x_0,n-1})^{(i)}(x_0) = 0 \end{align*}

Sei \(c\), so dass gilt

\begin{align*} f(x) - T_{x_0,n-1}(x) = c \cdot (x - x_0)^n \end{align*}

Insbesondere existiert ein \(c\) wegen \(x - x_0 \neq 0\) Für die Hilfsfunktion

\begin{align*} h(x) \coloneqq f(x) - T_{x_0,n-1}(x) - c (x - x_0)^n \end{align*}

gilt:

\begin{align*} h^{(1)}(x_0) = ... = h^{(n-1)}(x_0) = 0 \\ h(x) = 0 \\ h(x_0) = 0 \end{align*}

und \(h\) ist \(n\)-mal differenzierbar zwischen x Durch \(n\)-fache Anwendung von Satz von Rolle folgt, dass ein \(\xi_1,...,\xi_n \in I\) existiert mit

\begin{align*} h^{(k)}(\xi_k) = 0 \end{align*}

Sei \(\xi_n = \xi\) dann folgt:

\begin{align*} h^{(n)}(\xi) =& 0 \\ =& f^{(n)}(\xi) - 0 - c \cdot n! \\ f^{(n)}(\xi) =& c \cdot n! \\ \frac{f^{(n)}(\xi)}{n!} =& c \end{align*}
