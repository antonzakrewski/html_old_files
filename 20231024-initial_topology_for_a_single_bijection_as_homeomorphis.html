initial topology for a single bijection as homeomorphism

1. Proposition

Let \((X, \mathcal{T}_X)\) be a topological space, \(Y\) a set and

\begin{align*} f: Y \rightarrow X \end{align*}

an Bijection For the topological space \((Y, \mathcal{T}_Y)\) with the initial topology, the map

\begin{align*} f: (Y, \mathcal{T}_Y) \rightarrow (X, \mathcal{T}_X) \end{align*}

is a homeomorphism
