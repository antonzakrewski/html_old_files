rekursive definition der menge freien Variablen

1. Definition

Sei \(A\) eine Formel bzw. ein Term

Dann ist die menge der freien Variablen \(\mathrm{fv}(A)\) rekursiv definiert durch

  1. für eine Konstante \(c\) ist \(\mathrm{fv}(c) = \emptyset\)
  2. für eine variable \(x\) ist \(\mathrm{fv}(x) = \{x\}\)
  3. für eine \(n\)-stelliges Funktionssymbol \(f^{(n)}\) und \(f^{(n)}(t_1,...,t_n)\) ist \(\mathrm{fv}(f^{(n)}(t_1,...,t_n)) = \bigcup_{i =1}^n \mathrm{fv}(t_i)\)
  4. für ein \(n\)-stelliges Relationssymbol \(P\) ist \(\mathrm{fv}(P(t_1,...,t_n)) = \bigcup_{i = 1}^n \mathrm{fv}(t_i)\)
  5. für \(\circ \in \{ \rightarrow, \land, \lor\}\) und Formeln \(A,B\) ist \(\mathrm{fv}(A \circ B) = \mathrm{fv}(A) \cup \mathrm{fv}(B)\)
  6. für \(\square \in \{\exists, \forall\}\) und eine Formel \(A\) ist \(\mathrm{fv}(\square_x A) = \mathrm{fv}(A) \setminus \{x\}\)
