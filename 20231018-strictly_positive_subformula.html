strictly positive subformula

1. Definition

Sei \(A\) eine Formel. Dann ist eine strikt positive Teilformel von \(A\) definiert durch

  1. \(A\) ist eine strikt positive Teilformel
  2. Falls \(B \lor C\) oder \(B \land C\) eine strikt positive Teilformel von \(A\) ist, so auch \(B,C\)
  3. Wenn \(\forall_x B(x), \exists_x B(x)\) eine strikt positive Teilformel von \(A\) ist, so auch \(B\)
  4. Wenn \(B \rightarrow C\) eine strikt positive Teilformel von \(A\) ist, so auch \(C\)
