Wurzel-Kriterium für Konvergenz

1. Satz

Sei $ ∑n=1∞ an$ eine komplexe Reihe Existiert ein \(0 < q < 1\), s.d.

\begin{align*} \sqrt[n]{\vert a_n \vert} \leq q \end{align*}

für fast alle \(n\) gilt, so konvergiert \(\sum_{n=1}^{\infty} a_n\) absolut

2. Beweis

sei o.B.d.A.

\begin{align*} \sqrt[n]{\vert a_n \vert} \leq q \end{align*}

so folgt:

\begin{align*} \vert a_n \vert \leq q^n \end{align*}

Daraus folgt, dass die Geometrische Reihe eine konvergente Majorante ist.
