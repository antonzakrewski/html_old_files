Äquivalenz: Transponierte Matrix und Reguläre Matrix

1. Satz

Sei eine Matrix. Dann sind folgende Aussagen äquivalent:

  • ist eine Reguläre Matrix
  • die Transponierte Matrix ist eine Reguläre Matrix

Insbesondere folgt, dass alle Äquivalente Charakterisierungen auch äquivalent zur Transponierte Matrix als Reguläre Matrix sind
