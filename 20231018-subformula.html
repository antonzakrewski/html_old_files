1. Definition

Sei \(A\) eine Formel. Eine Teilformel von \(A\) ist definiert durch eine der 3 Möglichkeiten

  1. \(A\) selbst ist eine Teilformel
  2. Für eine Teilformeln \(B \circ C\) von \(A\) für \(\circ \in \{ \rightarrow, \lor, \land\}\) sind auch \(B,C\) teilformeln
  3. Falls \(\forall_x B(x), \exists_x B(x)\) eine Teilformel von \(A\) ist, so auch \(B\).
