Stetige Funktionenraum als direkte Summe von geraden und ungeraden Funktionen

1. Satz

Sei \(V\) der reelle Vektorraum der stetigen, beschränkten Funktionen \(f: \mathbb{R} \rightarrow \mathbb{R}\) gegeben sowie \(U\) die Menge der ungerade Funktionen und \(G\) die Menge der geraden Funktionen. Dann gilt

\begin{align*} V = U \bigoplus G \end{align*}

2. Beweis

2.1. summe

Sei \(f' \in V\) eine beliebige Funktion. Wir können \(f \coloneqq f - f(0)\) setzen, damit \(f(0) = 0\) gilt. Sei im folgenden die ungerade Funktion \(g_u\) und die gerade Funktion \(g_g\) definiert als:

\begin{align*} g_u(x) \coloneqq \begin{cases} \frac{1}{2} f(x) & \mbox{if } x \leq 0 \\ -\frac{1}{2} f(-x) & \mbox{if } x > 0 \end{cases} g_g \coloneqq& \begin{cases} \frac{1}{2} f(x) & \mbox{if } x \leq 0 \\ \frac{1}{2} f(-x) & \mbox{if } x > 0 \\ \end{cases} \end{align*}

Somit gilt

\begin{align*} g_u + g_g =& \begin{cases} f(x) & \mbox{if } x \leq 0 \\ 0 & \mbox{else } x > 0 \\ \end{cases} \end{align*}

Analog erhält man für die andere Hälfte funktionen \(h_u, h_g\) und es folgt:

\begin{align*} f' =& h_u + h_g + g_u + g_g + f(0) \\ =& (h_u + g_u) + (h_g + g_g + f0) \end{align*}

2.2. direkt

Sei \(f \in (G \cap U)\) und \(x \in \mathbb{R}\) beliebig, so gilt \(f(x) = f(-x) = -f(x)\). Damit folgt dann \(f(x) = 0\) und somit auch \(f = 0\)
