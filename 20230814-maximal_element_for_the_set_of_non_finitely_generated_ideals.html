maximal element for the set of non finitely generated ideals

1. Proposition

Let \(A\) be a commutative ring, \(S = \{\mathfrak{a} \subseteq A \vert \mathfrak{a} \text{ not finitely generated ideal} \}\) nonempty Then \(S\) contains a maximal element.

2. Proof

Corollary of:

  • Lemma von Zorn
  • union of a chain of not finitely generated ideals not finitely generated
