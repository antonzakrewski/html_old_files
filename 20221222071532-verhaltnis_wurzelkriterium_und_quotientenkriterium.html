Verhältnis: Wurzelkriterium und Quotientenkriterium

1. Satz

Das Wurzel-Kriterium für Konvergenz ist impliziert das Quotientenkriterium für Konvergenz in Abhängigkeit vom Limes Superior, aber nicht umgekehrt. Dabei ist das Wurzelkriterium oft einfacher anwendbar.

2. Beweis

Es sei \((a_n)_{n \in \mathbb{N}}\) eine Folge positiver reeller Zahlen. Dann folgt:

\begin{align*} \lim_{n \to \infty}\left(\sup \sqrt[n]{a_n} \right) \leq \lim_{n \to \infty}\left(\sup \frac{a_{n+1}}{a_n}\right) \end{align*}

o.B.d.A. sei \(\lim_{n \to \infty}\left(\sup \frac{a_{n+1}}{a_n}\right) =: C < \infty\) Sei \(\varepsilon > 0\) beliebig, dann folgt, dass ein \(n_{\varepsilon} \in \mathbb{N}\) existiert, sodass für \(\forall n > n_{ \varepsilon}\) gilt:

\begin{align*} \frac{a_{n+1}}{a_{n}} < C + \varepsilon \end{align*}

Sei \(n_{\varepsilon} < n\), dann folgt

\begin{align*} a_n < a_{n-1}(C + \varepsilon) < a_{n-2} (C + \varepsilon)^2 < ... < a_{n_{\varepsilon}} (C + \varepsilon)^{n - n_{\varepsilon}} \end{align*} \begin{align*} \sqrt[n]{a_{n}} < \sqrt[n]{a_{n_{\varepsilon}}} (C + \varepsilon)^{1 - \frac{n_{\varepsilon}}{n}} \end{align*}

Es folgt:

\begin{align*} \lim_{n \to \infty}\left(\sqrt[n]{a_{n_{\varepsilon}}}\right) =& 1 \\ \lim_{n \to \infty}\left((C + \varepsilon)^{1 - \frac{n_{\varepsilon}}{n}}\right) =& C + \varepsilon \end{align*}

Daraus folgt:

\begin{align*} \exists n'_{\varepsilon} > n_{\varepsilon} : \sqrt[n]{a_n} < C + 2 \varepsilon \forall n > n'_{\varepsilon} \end{align*}

Daraus folgt:

\begin{align*} \lim_{n \to \infty}\left( \sup \sqrt[n]{a_n}\right) \leq C + 2 \varepsilon \forall \varepsilon > 0 \Rightarrow lim_{n \rightarrow \infty} \sup \sqrt[n]{a_n} \leq C \end{align*}
