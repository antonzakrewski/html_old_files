n = n (Anordungsaxiome)

1. Satz

\begin{align*} n = n \end{align*}

2. Beweis

  • Beweis durch Widerspruch
  • Trichotomie (Axiom)
  • Definition: Relationen
\begin{align*} a_1 > a_2 \Rightarrow a_2 < a_1 \end{align*}

Sei:

\begin{align*} n = a_1 = a_2 \\ \Rightarrow n > n \Leftrightarrow n < n \text{\Lightning} \end{align*}
