generic point and open set

1. Proposition

Let \((X, \mathcal{T})\) be a topological space and \(x \in X\).

TFAE:

  1. \(x\) is a generic point
  2. any nonempty open set contains \(x\)

2. Proof

2.1. 1) \(\implies\) 2)

2.2. 2) \(\implies\) 1)
