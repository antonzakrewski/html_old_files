irreducible space and open set as connected set

1. Proposition

Let \((X, \mathcal{T})\) be a topological space.

TFAE:

  1. \(X\) is an irreducible space
  2. any open set is connected

2. Proof

2.1. 1) \(\implies\) 2)

Proof by contraposition Suppose there exists an \(O\) open, \(O_1,O_2 \neq \emptyset\) and \(O_1 \sqcup O_2 \supseteq O\). Then \(X \setminus O_1, X \setminus O_2 \subseteq X\) are closed with

\begin{align*} (X \setminus O_1) \cup (X \setminus O_2) =& X \setminus (O_1 \cap O_2) \\ =& X \setminus \emptyset =& X \end{align*}

Furthermore \((X \setminus O_1), (X \setminus O_2) \neq X\)

2.2. 2) \(\implies\) 1)

Proof by contraposition Suppose \(X\) is not irreducible, then there exist closed sets \(A_1,A_2 \subseteq X\) such that \(A_1 \cup A_2 = X\). Then \(X \setminus A_1, X \setminus A_2 \subseteq X\) are open and further disjoint, because of \(A_1 \cup A_2 = X\). Thus \(X_1 \setminus A_1 \sqcup X_1 \setminus A_2\) is open and disconnected
