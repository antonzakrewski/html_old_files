density and open sets

1. Proposition

Let \((X, \mathcal{T})\) be a topological space. A subset \(D \subseteq X\) is dense in \(X\) if and only if for each nonempty open set \(A\) the intersection is nonempty

2. Proof

2.1. a)

Suppose \(O \cap D = \emptyset\). Then \(X \setminus O\) is open and \(\overline{D} \subseteq \overline{X \setminus O} = X \setminus O\) holds true (see Abschluss einer abgeschlossenen Menge). This contradicts \(\overline{D} = X\)

2.2. b)

Suppose \(A\) is a nonempty open set. Then for \(a \in A\) there exists a disjoint neighbourhood and therefore \(a \not\in \overline{D}\).
