Dedekindsche Schnittaxiom

1. Definition

Sei \((X,<)\) eine totalgeordnete Menge. Dann erfüllt \(X\) das Schnittaxiom, falls für eine Partition \(A \sqcup B\) mit \(A < B\) (siehe: Vergleich von Mengen) jeweils ein Supremum bzw. Infimum existiert, so dass gilt

\begin{align*} \sup A = \inf B \end{align*}
