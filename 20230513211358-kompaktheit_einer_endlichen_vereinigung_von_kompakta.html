Kompaktheit einer endlichen Vereinigung von Kompakta

1. Satz

Sei \((X, \mathcal{T})\) ein topologischer Raum und \(K_i\) für \(i \in \{1,..,n\}\) kompakte Mengen Dann ist

\begin{align*} K \coloneqq \bigcup_{i = 1}^{n} K_i \end{align*}

ebenfalls Kompakt

2. Beweis

Sei \((O_j)\) eine offene Überdeckung von \(K\), so ist \((K_i \cap O_j)\) eine offene Überdeckung von \(K_i\). Nach Annahme existiert jeweils für endlich viele \(K_i\) jeweils eine endliche Teilüberdeckung \((K_i \cap O_j')\). Indem man für jedes \(K_i\) die \(O_j'\) wählt, erhält man damit die gewünschte endliche Teilüberdeckung.
