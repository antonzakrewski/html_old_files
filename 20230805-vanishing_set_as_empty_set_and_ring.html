Vanishing set as empty set and Ring

1. Proposition

Let \(A\) be a commutative ring and \(\mathrm{Spec}(A)\) zariski spectrum equipped with the Zariski topology and \(B \subseteq B\) TFAE:

  1. \((B) = A\)
  2. \(V(B) = \emptyset\)

see: Vanishing set

2. Proof

2.1. 1) \(\implies\) 2)

Let \(\mathfrak{p} \in V(B)\), then

2.2. 2) \(\implies\) 1)

Proof by contraposition: Suppose \((B) \neq A\), then there exists a maximal ideal \(\mathfrak{m}\) (see: existence of a maximal ideal for a strict ideal). Furthermore, by construction, \(\mathfrak{m} \in V(B)\). Hence \(V(B) \neq \emptyset\) weak Nullstellensatz
