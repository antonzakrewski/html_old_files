zariski topology on an affine space and Hausdorff-space

1. Proposition

Let \(K\) be a field, \(\mathbb{A}_K^n\) the affine space with the zariski topology.

TFAE:

  1. \(\vert K \vert = \infty\)
  2. \(\mathbb{A}_K^n\) is not Hausdorff

2. Proof

2.1. 1) \(\implies\) 2)

corollary of:

  • simple affine line has cofinite topology
  • cofinite topology not hausdorff for an infinite set
  • Vererbung der T2-Eigenschaft \(\Rightarrow\) contradiction

2.2. 2) \(\implies\) 1)

Suppose \(K\) is finite, then \(\vert \mathbb{A}_K^n \vert\) is finite. Furthermore, each point is closed as algebraic set generated by \((x_1 - \alpha_1,...,x_n - \alpha_n)\) thus we have the discrete topology, which is Hausdorff-Raum, contradiction
