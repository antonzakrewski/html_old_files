Zettelkasten Software - Anforderungen

1. Liste

1.1. Elementar

  • Verlinkungen
  • irgendein Markup
    • LaTeX Integration
  • offenes dateiformat -> unabhängigkeit von (weiterentwicklung) Software

  • Graphische Darstellung
  • Metadaten:
    • tags
    • datum
  • Publishing
  • Open Source \(\rightarrow\) mehr freiheiten für:
    • Plugins
    • eigene Configurationen
  • Guter Editor
  • Backlinks
