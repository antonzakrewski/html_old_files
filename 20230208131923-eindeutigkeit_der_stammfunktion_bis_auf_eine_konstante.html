Eindeutigkeit der Stammfunktion bis auf eine Konstante

1. Satz

Die Stammfunktion ist eindeutig bis auf eine Kontsante

2. Beweis

Sei eine Funktion mit Stammfunktionen , so folgt für die Funktion

Daraus folgt aus der Ableitung gleich 0 eine konstante Funktion (siehe: Ableitung und konstante Funktion) und damit für ein konstantes
