Eindeutigkeit der Stammfunktion bis auf eine Konstante

1. Satz

Die Stammfunktion ist eindeutig bis auf eine Kontsante

2. Beweis

Sei \(f\) eine Funktion mit Stammfunktionen \(F_1(x),F_2(x)\), so folgt für die Funktion

\begin{align*} h(x) \coloneqq& F_1(x) - F_2(x) \\ h'(x) =& 0 \end{align*}

Daraus folgt aus der Ableitung gleich 0 eine konstante Funktion (siehe: Ableitung und konstante Funktion) und damit für ein konstantes \(c:\)

\begin{align*} F_1(x) - F_2(x) = c\\ F_1(x) = F_2(x) + c \end{align*}
