Satz von Mazurkiewicz

1. Satz

Sei \(X\) ein polnischer Raum und \(A \subseteq X\). Dann ist \(A\) polnisch g.d.w. \(A\) eine G-Delta Menge in \(X\) ist

2. Beweis

2.1. a)

folgt aus den Aussagen:

  1. Vererbung von polnischen Räumen auf offene Unterräume
  2. Schnitt abzählbar vieler, polnischer Unterräume als polnischer Unterraum

2.2. b)

Sei \(A\) polnisch, so wähle man eine vollständige Metrik \(d_A, d_X\). Dann ist die Inklusionsabbildung ein Homeomorphismus auf das Bild. Nach dem Satz von Lavrentiell existiert ein \(h^*: A^* \rightarrow B^* \subseteq X\), so dass \(B^*\) eine G-Delta Menge in \(X\) ist. Es existiert aber bloß die triviale Forsetzung mit \(A^* = A\), also ist \(\iota[A^*] = B^* = A^*\) eine G-Delta Menge.
