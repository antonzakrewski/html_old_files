Bernoullische Ungleichung

1. Satz

\begin{align*} (1 + x)^n \geq 1 + nx \quad n \in \mathbb{N}, x \geq -1 \end{align*}

Mit gleichheit g.d.w. \(n = 0\) oder \(n = 1\) oder \(x = 0\)

2.1. Induktionsanfang

\begin{align*} (1 + x)^0 = 1 = 1 + 0 \cdot x\\ (1 + x)^1 = 1 + x = 1 + 1 \cdot x \end{align*}

2.2. Induktionsschritt

\begin{align*} (1 + x)^{n+1} =& (1 + x) \cdot (1+x)^n \\ =& (1 + x)^n + (x)(1+x)^n \\ 1 + (n+1)x =& 1 + nx + x \\ (1 + x)^n \geq 1 + nx \\ (x)(1+x)^n > x % (x)(1+x)^n =& \begin{cases} % \binom{ & \mbox{if } x > 0 \\ % \sum_{k=0}^n \binom{n}{k} 1 & \mbox{if } x = 0 \\ % & \mbox{if } -1 \geq x < 0 \\ \end{cases} \end{align*}
