lokalkompaktes Bild einer stetigen, offenen, surjektiven Abbildung

1. Satz

Seien \((X, \mathcal{T}_X)\) und \((Y, \mathcal{T}_Y)\) topologische Räume und \(f: X \rightarrow Y\) eine stetige, offene surjektion. Sei \(X\) lokalkompakt, so ist \(Y\) ebnefalls lokalkompakt.

2. Beweis

Sei \(O_{i}\) eine offene Überdeckung von \(Y\), so ist \(U_i \coloneqq f^{-1}[O_i]\) nach Annahme der Stetigkeit ebenfalls eine offene Überdeckung von \(X\). Sei \(y \in Y\) beliebig, so existiert ein \(x \in X\) mit \(f(x) = y\). Da \(X\) lokalkompakt ist, existiert eine Kompakte Umgebung \(K\) von \(x\) Damit ist \(f[K]\) eine Umgebung von \(f(x)\) (vgl. offene Abbildungen erhalten Umgebungen) und Kompakt (vgl. Kompaktheit des Bildes einer kompakten Menge einer stetigen Funktion)
