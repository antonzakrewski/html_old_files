Satz von Rolle

1. Satz

Sei \([a,b] \in \overline{\mathbb{R}}\) für \(-\infty \leq a < b \leq +\infty\) mit \(f:[a,b] \rightarrow \mathbb{R}\) eine stetige, differenzierbare Funktion, so dass gilt:

\begin{align*} f(a) = f(b) \end{align*}

so folgt, dass \(x_0 \in [a,b]\) existiert, so dass gilt

\begin{align*} f'(x_0) = 0 \end{align*}

siehe Ableitung

2. Beweis

Sei o.B.d.A. \(f\) nicht konstant (sonst trivial), so folgt auch, dass o.B.d.A. ein \(y\) existiert, so dass gilt \(f(a) = f(b) \neq y\) und \(\exists x \in (a,b) : f(x) = y\) Nach Zwischenwertsatz existieren somit für z.B. \(y' = \frac{f(a) + y}{2}\) verschiedene \(x_1,x_2 \in (a,b) = \mathbb{R}\), so dass gilt: \(f(x_1) = f(x_2)\). Damit haben wir den Fall \([-\infty, \infty]\) auf \([x_1,x_2]\) reduziert, das heißt wir spätestens hier haben wir hier ein folgenkompaktes Intervall und nach dem Satz vom Minimum und Maximum folgt, dass ein lokales Extremum existiert. Nach der Nullstelle der Ableitung und lokales Extremum folgt somit auch, dass ein \(x_0 \in [x_1,x_2]\) existiert, so dass gilt: \(f(x_0) = 0\)

2.1. Alternativ:

Annahme einer stetige Fortsetzung einer Funktion für \(-\infty, \infty\) und dann stetigkeit
