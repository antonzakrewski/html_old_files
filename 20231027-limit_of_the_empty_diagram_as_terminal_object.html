limit of the empty diagram as terminal object

1. Proposition

Let \(\mathcal{C}\) be a category. Then a terminal object, if it exists, is a limit of the empty diagram

\begin{align*} \mathcal{F}: \empty \rightarrow \mathcal{C} \end{align*}

2. Proof

is dual to colimit of the empty diagram as initial object
