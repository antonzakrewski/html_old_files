algebraic set and order

1. Proposition

Let \(K\) be a field and \(\mathbb{A}_{K}^n\) the affine space. Then for subsets \(S_1 \subseteq S_2 \subseteq K[X_1,...,X_n]\) we conclude

\begin{align*} V(S_2) \subseteq V(S_1) \end{align*}

2. Proof

Defintions: If \((p_1,...,p_n) \in V(S_2)\), then for any \(f \in S_2\) it follows, that \(f(p_1,...,p_n) = 0\). Thus, it also holds for \(f \in S_1\).
