Translationsinvarianz einer von einer Norm induzierten Metrik

1. Satz

Sei \((V,\| \|)\) ein normierter Raum, so ist die induzierte Metrik \(d\) auf \(V\) translationsinvariant, d.h. es gilt:

\begin{align*} d(v_1,v_2) = d(v_1 + m, v_2 + m) \end{align*}

2. Beweis

folgt durch Einsetzen:

\begin{align*} \| v_1 + m - (v_2 + m) \| =& \| v_1 - v_2 \| \\ =& d(v_1,v_2) \end{align*}
