Closed set in a subspace topology as intersection with a closed set

1. Proposition

Let \((X, \mathcal{T})\) be a topological space and \(Y \subseteq X\) the subspace topology. A set \(A \subseteq Y\) is closed if and only if a closed set \(D \subseteq X\) exists, such that

\begin{align*} A = D \cap Y \end{align*}

2. Proof

2.1. a)

We know, that \(Y \setminus A\) is open, i.e. there exists an open set \(O \subseteq X\) such that \(Y \cap O = Y \setminus A\). We also conclude, that \(X \setminus O\) is closed and it follows, that \((X \setminus O) \cap Y = Y \setminus (Y \cap O) = Y \setminus (Y \setminus A) = A\)

2.2. b)

Suppose \(A \subseteq X\) closed, then \(X \setminus A\) is open and therefore \(Y \cap (X \setminus A) = Y \setminus (Y \cap A)\) is open. It follows, that \(Y \setminus (Y \setminus (Y \cap A)= Y \cap A\) is closed.
