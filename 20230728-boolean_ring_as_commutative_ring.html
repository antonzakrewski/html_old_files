Boolean ring as commutative ring

1. Proposition

Let \(R\) be a Boolean ring, then \(R\) is a commutative ring

2. Proof

Let \(r_1,r_2 \in R\), then

\begin{align*} (r_1 + r_2)^2 - (r_1 + r_2) =& 0 \\ r_1^2 + r_2^2 + r_1r_2 + r_2r_1 - r_1 - r_2 =& 0 \\ r_1 - r_1 + r_2 - r_2 + r_1r_2 + r_2r_1 =& 0 && \vert - r_2r_1\\ r_1r_2 =& -r_2r_1 \\ \end{align*}

Because \(x = -x\) (see: boolean ring and additive inverse) we conclude

\begin{align*} r_1r_2 = r_2r_1 \end{align*}
