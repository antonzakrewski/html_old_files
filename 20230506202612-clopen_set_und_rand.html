Clopen set und Rand

1. Satz

Sei \((X, \mathcal{T})\) ein topologischer Raum. Eine Teilmenge \(A \subseteq X\) ist clopen g.d.w. für den Rand gilt:

\begin{align*} \partial A = \emptyset \end{align*}

2. Beweis

Sei \(A\) clopen, so ist das äquivalent zu \(\overline{A} = A = \mathring{A}\). Dies ist äquivalent zur Aussage \(\partial A = \overline{A} \setminus \mathring{A} = \emptyset\)
