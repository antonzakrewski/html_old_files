Intersection of a relatively compact set as relative compact set

1. Proposition

Let \((X, \mathcal{T})\) be a topological space, \(R \subseteq X\) a relatively compact set and \(V \subseteq X\) an arbitrary set Then \(R \cap V\) is also relatively compact

2. Proof

By closure of an intersection it holds, that

\begin{align*} \overline{R \cap V} \subseteq \overline{R} \end{align*}

Furthermore, closed subsets of compact spacese are compact, hence \(\overline{R \cap V}\) is compact and thus \(R \cap V\) is relatively compact.
