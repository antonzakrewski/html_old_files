vektorwertige glatte Funktion auf einer glatten Mannigfaltigkeit

1. Definition

Eine Funktion \(f: M \rightarrow \mathbb{R}^k\) auf einer glatte Mannigfaltigkeit \((M, [\mathcal{A}])\) ist eine glatte Funktion, falls dass für einen (oder äquivalent alle) differenzierbaren Atlanten \(\mathcal{A}\) gilt: für jede Karte \((U,\varphi) \in \mathcal{A}\) ist

\begin{align*} f \circ \varphi^{-1}: \varphi(U) \rightarrow \mathbb{R}^k \end{align*}

eine glatte Funktion
