number of connected components and connected Space

1. Proposition

Let \((X, \mathcal{T})\) be a topological space.

TFAE:

  1. \(X\) is connected
  2. for the set of connected components \(\pi(X)\) it holds, that
\begin{align*} \vert \pi(X) \vert = 1 \end{align*}

2. Proof

definitions note:

  • empty space not connected
