Riemannsche Mannigfaltigkeit

1. Definition

Eine riemannsche Mannigfaltigkeit \(M\) ist eine glatte Mannigfaltigkeit \(M\) zusammen mit einer riemannschen Metrik \(g\)

einem schnitt \(g\) von

\begin{align*} TM^* \otimes TM^* \end{align*}

so dass für jeden Punkt \(p \in M\) die Bilinearform \(g(p) \in TM^* \otimes TM^*\) ein reelles Skalarprodukt, auch ist vgl. Tensorn von Vektorbündel
