Gleichmäßige Stetigkeit einer stetigen Funktion von einer Kompakten Menge

1. Satz

Sei \(f: D \rightarrow N\) eine stetige Funktion auf zwei metrischen Räumen und \(D\) zusätzlich ein folgenkompakte Menge Dann ist \(f\) gleichmäßig stetig

2. Beweis

Beweis durch Widerspruch Angenommen \(f\) ist nicht gleichmäßig stetig, so folgt, dass ein \(\varepsilon_0 > 0\) existiert, so dass gilt: Es existieren zwei Folgen \((z_n), (z_n')\) in \(D\), so dass gilt

\begin{align*} \forall n \in \mathbb{N} \begin{cases} 0 < \vert z_n - z_n' \vert < \frac{1}{n} \\ \vert f(z_n) - f(z_n') \vert \geq \varepsilon_0 \end{cases} \end{align*}

Aus der Folgenkompaktheit gilt, dass eine Teilfolge mit \(z_{n_k} \rightarrow \zeta\) existiert. Zudem konvergiert \(z_{n_k}' \rightarrow \zeta\) wegen: Häufungspunkt und Existenz einer konvergenten Folge in erstabzählbaren Räumen Wegen der Dreiecksungleichung gilt:

\begin{align*} \vert z_{n_k}' - \zeta \vert =& \vert z_{n_k}' - z_{n_k} + z_{n_{k}} - \zeta \vert \\ \leq& \vert z_{n_k}' - z_{n_k} \vert + \vert z_{n_k} - \zeta \vert \rightarrow 0 \end{align*}

und aufgrund der Folgenstetigkeit gilt:

\begin{align*} \vert f(z_{n_k}) - f(z'_{n_k}) \vert \rightarrow \vert f(\zeta) - f(\zeta) \vert = 0 \end{align*}

trotz \(z_{n_k} - z_{n_k}' > 0\) was ein Widerspruch ist zur Annahme

\begin{align*} \vert f({z_n}) - f(z_n') \vert \geq \varepsilon_0 \forall n \end{align*}

und damit auch für alle \(n_k\)
