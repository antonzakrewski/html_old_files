Erweiterung eines Ordnungsisomorphismus von dichten, abzählbaren Teilmengen der reellen Zahlen auf ℝ

1. Satz

Seien \(A,B \subseteq \mathbb{R}\) dichte, abzählbare Mengen mit einem Ordnungsisomorphismus \(f: A \rightarrow B\), so ist Dann ist

\begin{align*} f': \overline{A} \rightarrow \overline{B} \\ x \mapsto \sup_{a \leq x} f(a) \end{align*}

ein Homeomorphismus auf den Unterraumtopologien \(\overline{A}, \overline{B}\).

2. Beweis

2.1. injektiv

Seien \(a,a' \in \overline{A}\) verschieden, so existiert aufgrund der Dichtheit von \(A\) ein \(\tilde{a}, \tilde{a}' \in A\) mit \(a < ... < \tilde{a} < ... < \tilde{a}' < ... < a'\). Daraus folgt \(f(a) < ... < f(\tilde{a}') < ... < f(a')\).

2.2. surjektiv

Sei \(b \in \overline{B}\) mit o.B.d.A. \(b \not \in B\), \(b\) nicht Maximum/Minimum so existiert nach Annahme Folgen \(b_i^+ \nearrow b, b_i^- \searrow b\) mit $bi ∈ B"a Damit folgt \(\lim_{i \to \infty}\left(f^{-1}(b_i^{-})\right) \eqqcolon a^{-}\), da \(f^{-1}(b_i)\) monoton und beschränkt ist und damit konvergiert. Analog für \(a^{+}\) und wir zeigen, dass \(a^- = a^+\). Angenommen nicht, so existiert wegen der Dichtheit ein \(a' \in (a^{-1},a^{+})\) mit o.B.d.A. \(f(a') \geq b\). Dabei existiert ein \(a'' \in (a'',a^{ +})\)

Für das Maximum/Minimum \(b\), sofern es existiert, reicht dabei eine Folge, die gegen \(b\) konvergiert

2.3. stetig

Sei \((b,b') \subseteq \overline{B}\) gegeben, so ist wegen der Ordnungserhaltung \(f^{-1}[(b,b')] = (f^{-1}(b),f^{-1}(b'))\) und damit offen

2.4. offene Abbildung

Sei \((a,a') \subseteq \overline{A}\), so gilt wegen der Ordnungserhaltung: \(f[(a,a')] = (f(a),f(a'))\).
