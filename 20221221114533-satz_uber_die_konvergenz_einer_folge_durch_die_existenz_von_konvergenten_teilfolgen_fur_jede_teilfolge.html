Konvergenz einer Folge durch die Existenz von konvergenten Teilfolgen für jede Teilfolge

1. Satz

Sei \(a_n\) eine Folge, so dass für jede Teilfolge \(b_n\) eine Teilfolge \(c_n \overset{\mathrm{\infty}}{ \rightarrow} a\) existiert. Dann konvergiert \(a_n\) gegen \(a\)

2. Beweis

Sei \(\varepsilon > 0\) beliebig, aber fest. Falls für fast alle \(a_n\) gilt: $ | a - an | < ε$, so konvergiert \(a_n\) Ansonsten existieren unendlich viele Folgenglieder, so dass sich eine Teilfolge \(a'_n\) existiert mit

\begin{align*} \vert a'_n - a \vert \geq \varepsilon && \forall n \in \mathbb{N} \end{align*}

Dies ist ein Widerspruch zur Annahme, dass sich für eine beliebige Folge eine konvergente Teilfolge konstruieren lässt.
