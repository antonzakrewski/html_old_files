power set functor

1. Definition

The power set functor \(\wp: \mathrm{Set} \rightarrow \mathrm{Set}\) is defined as: For sets \(A,B\) and a map \(f: A \rightarrow B\):

\begin{align*} \wp: A \mapsto& \mathcal{P}(A) \\ \wp: f \mapsto& \begin{cases} f: \mathcal{P}(A) & \rightarrow \mathcal{P}(B) \\ \hphantom{f:\, }X &\mapsto f[X] \end{cases} \end{align*}
