Existenz einer Diagonalisierung einer hermiteschen Matrix mit einer unitären Matrix

1. Satz

Sei \(V\) ein Prähilbertraum und \(A\) ein hermitesche Matrix, so existiert eine unitäre Matrix \(U\), so dass

\begin{align*} U \circ A \circ U^* \end{align*}

eine Diagonalmatrix ist.

2. Beweis

  • Existenz einer Diagonalisierung mit einer Unitären Matrix für einen normalen Endomorphismen
  • selbstadjungierter Endomorphismus als normaler Endomorphismus
  • selbst-adjungierte Abbildung und hermitesche Matrix
