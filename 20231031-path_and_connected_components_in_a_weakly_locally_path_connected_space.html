path and connected components in a weakly locally path connected space

1. Proposition

Let \((X, \mathcal{T})\) be a weakly locally path connected topological space. Then a path component is the same as connected component

2. Proof

\(X_i\) connected componente of \(X\)

\begin{align*} \pi_0(\coprod_{i \in I} X_i) \simeq \coprod_{i \in I} \pi_0(X_i) \end{align*}

reduced to following claim: \(X\) weakly locally path connceted and connected \(\Rightarrow X\) is path connected
