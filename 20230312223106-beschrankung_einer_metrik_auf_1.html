Beschränkung einer Metrik auf 1 mit einem Bruch

1. Satz

Sei \((M,d)\) ein metrischer Raum, so ist \(d'(x,y) \coloneqq \frac{d(x,y)}{1 + d(x,y)}\) ebenfalls eine Metrik, welche zudem auf \(1\) beschränkt ist.

2. Beweis

2.1. Symmetrie einer Metrik und Positive Definitheit (Metrik)

Symmetrie einer Metrik und Positive Definitheit (Metrik) folgen aus den Eigenschaften von \(d(x,y)\)

2.2. Dreiecksungleichung

Seien \(x,y,z \in M\), so folgt:

\begin{align*} d'(x,y) =& \frac{d(x,y)}{1 + d(x,y)} \\ =& \frac{d(x,y)(1 + (d(x,z))(1 + d(y,z))}{(1 + d(x,y))(1 + d(x,z))(1 + d(z,y))} \end{align*}

und

\begin{align*} \frac{d(x,z)}{1 + d(x,z)} + \frac{d(z,y)}{1 + d(z,y)} =& \frac{(1 + d(x,y))(1 + d(z,y)) d(x,z)}{(1 + d(x,y))(1 + d(x,z)(1 + d(z,y)))} + \frac{(1 + d(x,y))(1 + d(x,z)) d(z,y)}{(1 + d(x,y))(1 + d(x,z)(1 + d(z,y)))} \\ \end{align*}

Dabei gilt:

\begin{align*} d(x,y)(1 + (d(x,z))(1 + d(y,z)) \\ =& d(x,y)d(x,z)d(y,z) + d(x,y)d(y,z) + d(x,y)d(x,z) \\ (1 + d(x,y)) (1 + d(z,y)) d(x,z) + (1 + d(x,y))(1 + d(x,z))d(z,y) \\ =& 2d(x,y)d(z,y)d(x,z) + d(x,y)d(z,y) + d(x,y)d(x,z) + 2d(x,z)d(z,y) + d(x,z) + d(z,y) \end{align*}

Daraus folgt dann eingesetzt in die Gleichung oben:

\begin{align*} \frac{d(x,y)}{1 + d(x,y)} \leq \frac{d(x,z)}{1 + d(x,z)} + \frac{d(z,y)}{1 + d(z,y)} \end{align*}
