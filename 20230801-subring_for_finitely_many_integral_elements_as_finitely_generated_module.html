subring for finitely many integral elements as finitely generated module

1. Proposition

Let \(A \subseteq B\) be rings and \(\alpha_1,...,\alpha_n \in B\) integral elements over \(A\) Then the subring

\begin{align*} A[\alpha_1,...,\alpha_n] \end{align*}

is a finitely generated module
