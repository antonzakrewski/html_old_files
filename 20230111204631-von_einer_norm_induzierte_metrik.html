von einer Norm induzierte Metrik

1. Definition / Satz

Sei \(V\) ein semimetrischer Raum, dann ist \(V\) g.d.w. ein metrischer Raum bezüglich:

\begin{align*} d(x,y) \coloneqq \| x - y \| \end{align*}

wenn \(V\) ein normierter Raum ist

2. Beweis

  • von einer Halbmetrik induzierte Semimetrik

2.1. Positive Definitheit (Metrik)

2.1.1. a)

Sei \(d(x,y) = 0\) so folgt \(\| x - y \| = 0\) und wegen der Positiven Definitheit einer Norm auch \(x - y = 0\), wegen der Eindeutigkeit des inversen Elements einer Gruppe folgt \(x = y\)

2.1.2. b)

Sei eine induzierte Metrik gegeben, so folgt für \(d(x,y) = 0 \Rightarrow x = y\) und damit auch für \(\| x - y \| = 0 \Rightarrow x = y\)
