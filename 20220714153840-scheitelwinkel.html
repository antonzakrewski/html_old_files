Scheitelwinkel

1. Satz

Seien zwei sich schneidende Geraden g,h gegeben. Dann sind zwei gegenüberliegende Winkel gleich groß

2. Beweis

\begin{tikzpicture} \coordinate (g1) at (-3,2); \coordinate (g2) at (3,-2); \coordinate (h1) at (3,2) ; \coordinate (h2) at (-3,-2) ; \path[name path=p1] (g1) -- (g2); \path[name path=p2] (h1) -- (h2) ; \path[name intersections={of=p1 and p2, by=I}]; % \draw[] (g1) -- (I) -- (h2) pic [draw=blue, "$\alpha$", angle radius=1.1cm] {angle=g1--I--h2} ; % \draw[] (g1) -- (I) -- (h1) pic [draw=green, "$\alpha'$", angle radius=1cm] {angle=h1--I--g1} ; % \draw[] (g2) -- (I) -- (h1) pic [draw=green, "$\alpha'$", angle radius=1cm] {angle=h1--I--g1} ; \draw[] (g1) -- (g2); \draw[] (h1) -- (h2); \draw[] pic ["$\beta $", draw, angle radius=1.1cm,] {angle=g1--I--h2 } pic ["$\alpha $", draw, angle radius=0.9cm,] {angle=h2--I--g2 } pic ["$\alpha' $", draw, angle radius=1cm,] {angle=h1--I--g1 } ; \end{tikzpicture}

o.B.d.A. über den Winkel einer Gerade

\begin{align*} 180^\circ =& \alpha + \beta \\ 180^\circ =& \alpha' + \beta \\ \Rightarrow \alpha =& \alpha' \end{align*}
