path connectedness as equivalence relation

1. Proposition

Let \((X, \mathcal{T})\) be a topological space. Then the relation

\begin{align*} x \sim y :\Leftrightarrow x \in P(y) \end{align*}

where \(P(y)\) denotes the path component of \(y\)

2. Proof(sketch)

2.1. reflexive

  • constant path

2.2. symmetry

  • inverse path

2.3. tranistive

  • concatenation of paths
