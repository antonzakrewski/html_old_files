Doppelsumme

1. Definition

\begin{align*} \sum_{k_1=m_1}^{n_1} \sum_{k_2=m_2}^{n_2+1} a_{k_{1}k_2} \coloneqq \sum_{k_1=m_1}^{n_1} \left( \left( \sum_{k_2=m_2}^{n_2} a_{k_{1}k_2}\right) + a_{k_{1}n+1}\right) \\ % \sum_{m_1 \leq k_1 \leq n_1} \sum_{m_2 \leq k_2 \leq n_2 + 1} a_{k_1k_2} \coloneqq \sum_{k_1=m_1}^{n_1} \sum_{k_2=m_2}^{n_2+1} a_{k_{1}k_2} \end{align*}

2. Beispiele

\begin{align*} \sum_{1 \leq k \leq 2} \sum_{1 \leq h \leq 2} a_{kh} =& (a_{11} + a_{12}) + (a_{21} + a_{22}) % \sum_{1\leq k \leq 3,\, 1\leq h\leq 3} a_{kh} =& \sum_{1 \leq k \leq 3} \sum_{1 \leq h \leq 3} a_{kh} \\ % \sum_{1\leq k,h \leq 3} a_{kh} =& \sum_{1\leq k \leq 3,\, 1\leq h\leq 3} a_{kh} \end{align*}
