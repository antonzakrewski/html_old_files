Zusammenhang einer Ordnungstopologie

1. Satz

Sei \((X,<)\) eine totalgeordnete Menge und \((X, \mathcal{T})\) die Ordnungstopologie. Dann ist \(X\) zusammenhängend g.d.w. \(X\) das Dedekindsche Schnittaxiom erfüllt

2. Beweis

2.1. a) Dedekind => Zusammenhang

Seien \(A,B \subseteq X\) mit \(A \sqcup B = X\) und \(A,B \neq \emptyset\). Dann existieren \(a \in A\), \(b \in B\) mit o.B.d.A. \(a < b\). Betrachten wir die Zusammenhangskomponenten

2.2. b)

Angenommen \(X\) erfüllt nicht das Dedekindsche Schnittaxiom. Dann existiert eine offene Zerlegung \(A \sqcup B\) mit \(A < B\), so dass eine der beiden Fälle zutritt:

  1. Es existiert kein Supremum
  2. Das Supremum stimmt nicht mit dem Infimum überein

Für 1) ist \(\bigcup_{a \in A} (-\infty,a') = A\) mit \(a < a'\), so dass \(a' \in A\) liegt wohldefiniert, da kein Supremum existiert Für 2) gilt \(A = \{x \in X \vert x \leq \inf(B)\}, B = \{x \in X \vert x \geq \sup(B)\}\) eine offene Zerlegung, da \(\sup(A) < \inf(B)\) gilt.
