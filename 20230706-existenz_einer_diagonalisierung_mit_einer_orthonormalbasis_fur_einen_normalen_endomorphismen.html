Existenz einer Diagonalisierung mit einer Unitären Matrix für einen normale Matrix

1. Satz

Sei \(K^n\) ein endlich erzeugter Prähilbertraum und \(A \in K^{n \times n}\) eine normale Matrix. Dann existiert eine unitäre Matrix \(U\), so dass

\begin{align*} U A U^{*} \end{align*}

eine Diagonalmatrix ist
