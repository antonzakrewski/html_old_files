set functor from Top as left adjoint

1. Proposition

Given category set and category Top, the underlying-set functor is a left adjoint to the functor from Set to the indiscrete topology

2. Proof

Follows from

\begin{align*} \eta: \mathrm{Hom}_{\mathrm{Top}}(-, \mathrm{indisc}(-)) \rightarrow& \mathrm{Hom}_{\mathrm{Set}}(\mathrm{Set}(-),-) \\ (f: (X, \mathcal{T}) \rightarrow (Y, \{Y, \emptyset\})) \mapsto& (f: X \rightarrow Y) \end{align*}
