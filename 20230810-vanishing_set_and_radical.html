vanishing set and radical

1. Proposition

Let \(A\) be a commutative ring and \(\mathrm{Spec}(A)\) the Zariski spectrum equipped with the Zariski topology. Suppose \(\mathfrak{a}\) is an ideal, then it holds for the Vanishing set

\begin{align*} V(\mathfrak{a}) = V(\sqrt{\mathfrak{a}}) \end{align*}
