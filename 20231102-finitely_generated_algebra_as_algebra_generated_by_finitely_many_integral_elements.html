finitely generated Algebra as Algebra generated by finitely many integral Elements

1. Proposition

Let \(A,B\) be rings and \(B\) an A-Algebra for a ring-homomorphism \(\varphi: A \rightarrow B\).

TFAE:

  1. \(B\) is a finitely generated Algebra
  2. \(B\) is generated by finitely many integral elements \(b_1,...,b_n\)

2. Proof

2.1. 1) \(\implies\) 2)

corollary of:

  • integral element and finitely generated module

2.2. 2) \(\implies\) 1)

w.l.o.g. \(n = 1\), the general case follows from induction

by assumption, there exists an epimorphism \(\varphi: A[X_1,...,X_m] \rightarrow A(b)\) (see: integral element and finitely generated module) hence \((A[X_1,...,X_m])...[X_1,...,X_s]\) is a finitely generated module
