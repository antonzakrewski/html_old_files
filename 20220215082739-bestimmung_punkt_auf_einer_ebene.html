Bestimmung: Punkt auf einer Ebene

1. Satz

Ein Punkt liegt genau dann auf einer Ebene, wenn eine der folgenden Aussagen gilt:

1.1. Koordinatenform: Punkt auf einer Ebene

1.2. Normalenform: Punkt auf einer Ebene

1.3. Parameterform: Punkt auf einer Ebene
