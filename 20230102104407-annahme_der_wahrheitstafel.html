Annahme der Wahrheitstafel

1. Satz

Wahrheitstafeln nehmen Bivalenz und das Distributivgesetz an:

\begin{align*} (\nu(A) = 0 \lor \nu(A) = 1) \land (\nu(B) = 0 \lor \nu(B) = 1) \Leftrightarrow (\nu(A) = 0 \land \nu(B) = 0) \lor (\nu(A) = 0 \land \nu(B) = 1) ... \end{align*}
